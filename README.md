## ✨ Fitur

- Tampilan minimalis & rapi
- Tombol **Salin** untuk setiap metode
- Notifikasi toast saat berhasil copy
- Feedback tombol berubah jadi "Tersalin!"
- Klik area card juga bisa copy
- Responsive (cocok untuk HP)

## 🖥️ Cara Menggunakan

1. Buka file `index.html` di browser
2. Klik tombol **Salin** pada metode pembayaran yang diinginkan
3. Paste nomor di aplikasi transfer/e-wallet
